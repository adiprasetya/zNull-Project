## Usage

```bash
> su -c znull
> su -c znull [-s|set] [number]
> su -c znull [-r|reset]
> su -c znull [gg] [packagenameGG]
> su -c znull [-h|help|--help]
```

### Adaway AdBlocker
```bash
> su -c adaway
```

## Example

### Changing the znull.conf to PUBG MOBILE LITE.
```bash
> su -c znull -s 1
```
or
```bash
> su -c znull set 1
```


### Reset Guest with now znull.conf
```bash
> su -c znull -r
```
or
```bash
> su -c znull reset
```
